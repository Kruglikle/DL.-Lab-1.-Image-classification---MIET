## Структура

- `Fruit_veg_classification.ipynb` — основной ноутбук с пайплайном.
- `requirements.txt` — зависимости проекта.
- `.gitignore` — исключения для Git.
- `.env` — локальные секреты (например, Kaggle credentials). В Git не добавляется.

## Быстрый старт

1. Создайте виртуальное окружение.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Откройте ноутбук и запустите все ячейки: `Fruit_veg_classification.ipynb`.

## Примечания

- Крупные файлы (датасеты, чекпоинты) должны быть исключены из репозитория.
- Воспроизводимость обеспечивается запуском всех ячеек в `Fruit_veg_classification.ipynb`.
- Для доступа к Kaggle используйте `.env` с ключами, файл уже исключен из Git.
